# Jenkins CI/CD pipeline for a Dance School Management Service

A walking skeleton of a Full-stack web application built using React and Spring Boot. The EMS currently allows basic CRUD operations on students for testing purposes. Heart of this project is a proof of concept for a CI/CD pipeline using jenkins to test and deploy on GCP.
